# LanShare 交接文档 v3.3.0

## 当前状态

项目正常运行，NSFW（海外剧）功能已完成适配。v3.3.0 已完成文件列表 UI 一致性修复和每日增量备份改造。

## 本次完成

- 表格元数据、操作按钮、状态按钮统一使用 `Noto Serif SC` 墨韵主字体
- 列表和宫格操作按钮统一为 30px 高度，文字水平/垂直居中
- 状态按钮箭头参与 Flex 布局并禁止压缩，不再与文字重叠
- 默认关闭逐次上传备份，改为每天 23:55 创建增量快照
- 首次快照完整复制，后续未变化文件通过备份区内部硬链接复用
- 每日快照覆盖 `shared` 与 `shared_NSFW`，并生成 `.snapshot.json` 清单
- 恢复请求体、单文件上传大小限制并修复旧 NSFW 上传分支

## 备份说明

- 每日快照目录：`backup/daily/<快照时间>/`
- 旧的逐次上传备份不会自动删除，需要确认新机制稳定后再单独清理
- 默认不自动删除历史快照；如需保留最近 N 天，后续再增加保留策略
- 服务必须在计划时间保持运行；可用 `DAILY_BACKUP_RUN_ON_START=1` 手动补备份
- 完整项目迁盘时直接迁移整个 `share_web`，默认相对路径会自动跟随新位置
