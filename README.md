# 局域网文件服务器

一个零依赖的本地局域网文件服务器，支持：

- 浏览共享目录、上传文件、下载文件、新建文件夹
- 音频处理：裁剪、变速、倒放、合并，导出 WAV/MP3/FLAC/WebM
- 视频截帧：批量导入视频，帧标记、时间轴定位、批量导出截图
- 主题切换：日间/夜间模式渐变过渡
- 自定义背景、光标特效、拖拽字符

## 启动

```powershell
npm start
```

默认监听：

- `http://127.0.0.1:8080`
- `http://你的局域网IP:8080`

默认共享目录：当前目录下的 `shared`

## 可选环境变量

```powershell
$env:PORT=9000
$env:SHARED_DIR='D:\\共享文件'
npm start
```

## 说明

- 上传通过网页端读取文件后发送到本地服务，适合局域网内部使用。
- 服务默认只允许访问共享目录内的内容，做了路径越界限制。
- 如果局域网其他设备无法访问，通常是 Windows 防火墙没有放行对应端口。
